一、课程介绍

本套视频教程是 bui 的进阶教程，我们使用 bui 框架来开发一款玩 Android APP，采用真实的 API ,如果对 bui 还不了解的朋友可以看我的 基础视频或者直接在 bui 官网去学习http://easybui.com，本套视频是视频+笔记的形式「文字版和视频版，方便朋友们去查阅」，本人水平有限，难免会出现疏漏和问题，欢迎大家留言互动，我们共同学习，一起进步，

ps:由于本人也要工作所以时间有限录视频和写笔记两者加起来会耗费大量时间，所以不能保证笔记和视频一定是同步的「视频可能会先于笔记发行」，请大家谅解
